# Support

## Getting Help

### Documentation

Start with the [README](README.md) for installation and quick start guides.

### Community Support

- **GitHub Discussions**: Ask questions and share ideas
- **Issues**: Report bugs or request features

### Before Opening an Issue

1. ✅ Read the README and documentation
2. ✅ Search existing issues (open and closed)
3. ✅ Check the FAQ in README
4. ✅ Try the Lite version if you're having context/memory issues

### How to Ask Good Questions

- Describe what you're trying to achieve (not just what you're doing)
- Include the exact error message or behavior
- Mention your platform (ChatGPT, Claude, Hermes, Edge Gallery, OpenCode, etc.)
- Share the relevant part of your configuration or prompt

### Commercial Support

For enterprise support, custom integrations, or consulting, contact the maintainer via GitHub.
